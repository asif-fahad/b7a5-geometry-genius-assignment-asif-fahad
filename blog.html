1) What is DOM and what is the purpose of DOM?

Ans: DOM full form is Document Object Model. DOM is used for making a dynamic websites. We can add or delete HTML elements and CSS by using DOM by writing code in Javascript.

2) How will you select HTML elements using DOM? Name the DOM methods.

Ans: We can select HTML elements by using DOM. We can use "getElementsByTagName()" for selecting Tag elements, "getElementsByClassName()" for selecting Class elements, "getElementById()" for selecting Id elements, "querySelector()" for selecting first element that matches the selector, "querySelectorAll()" for selecting same selector.

3) What is event bubble?

Ans: Event bubble means when an element receives an event then it goes up to parent until it is stopped. It goes up to containing elements HTML tree.

4) What is a callback function and why will you use it?

Ans: A callback function is used for calling another function for executing a code. It is passed as an argument.
